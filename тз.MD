# Техническое задание

## Проект: FinFlow Analytics Platform

**Тип проекта:** Учебный enterprise-проект уровня Middle/Middle+

**Цель:** Разработать распределённую микросервисную платформу для управления заказами, платежами и аналитикой в режиме реального времени.

Проект должен демонстрировать практическое применение современных технологий, востребованных на рынке Java-разработки:

* Java 21
* Spring Boot 3
* Spring Security
* Spring Cloud
* PostgreSQL
* Redis
* Kafka
* ClickHouse
* Docker
* Kubernetes
* CI/CD
* Prometheus/Grafana
* Testcontainers
* OpenTelemetry

---

# 6. ClickHouse

## Таблица payment_events

```sql
CREATE TABLE payment_events
(
    event_id UUID,
    payment_id UUID,
    order_id UUID,
    user_id UUID,
    amount Decimal(12,2),
    status String,
    created_at DateTime
)
ENGINE = MergeTree
PARTITION BY toYYYYMM(created_at)
ORDER BY (created_at, user_id);
```

---

## Таблица order_events

```sql
CREATE TABLE order_events
(
    event_id UUID,
    order_id UUID,
    user_id UUID,
    amount Decimal(12,2),
    status String,
    created_at DateTime
)
ENGINE = MergeTree
```
